# 💰 Sistema Bancário em Java

Projeto simples de um sistema bancário feito em Java para praticar lógica de programação.

## 🚀 Funcionalidades
- Saque
- Depósito 
- Consulta de saldo

## 🛠️ Tecnologias
- Java
- Scanner (entrada de dados)

## 📚 Aprendizados
- Estrutura de repetição (for)
- Condições (if/else)
- Manipulação de variáveis

## ▶️ Como executar
1. Baixe o projeto
2. Compile o arquivo:
   javac Main.java
3. Execute:
   java Main
